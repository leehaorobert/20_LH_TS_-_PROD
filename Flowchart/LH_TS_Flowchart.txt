开始;
block 设定可选股票范围{
取得行业指数(16只);
计算行业指数的5日增长率;
按照增长率进行降序排序;
取得增长率排名倒数第二的行业指数;
取得该行业指数包含的所有股票;
取得所有股票的市值;
股票信息按照市值降序排序;
取得排名第一的股票;
while(该股票停牌)
  选取下一排名的股票;
生成可选股票清单;
}
block 生成卖出/买入清单{
  取得当前的股票持仓;
  if(当前股票停牌){
    继续持仓该股票(不处理);
  }
  else{
    if(当前股票涨停(>9%)){
      继续持仓该股票(不处理);
    }
    else{
      if(当前股票在可选股票清单){
        继续持仓该股票(不处理);
      }
      else
    }
  }
  生成卖出股票清单;
  生成买入股票清单;
}
block 执行卖出/买入操作{
  执行卖出操作;
  执行买入操作;
}
结束;
